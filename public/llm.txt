# Joyo - AI Gift Journey Builder

## Overview
Joyo is a free web platform that enables users to create emotional, interactive gift journeys using AI. These journeys are shareable experiences that combine storytelling, questions, media, and reveals to create memorable moments for special occasions.

## What is a Gift Journey?
A gift journey is an interactive sequence of screens that recipients experience one-by-one on their device. Each journey can include:
- Story screens with text and images
- Multiple choice questions
- Text input prompts
- Big reveal moments (with confetti animations)
- Full-screen media displays
- Custom themes and styling

## Key Features
- **AI-Powered Creation**: Describe your gift idea in natural language, and AI generates a complete journey with 5-8 screens
- **Visual Editor**: Drag-and-drop interface to reorder, edit, add, or delete screens
- **Image Integration**: Upload custom images or search Unsplash directly in the editor
- **Beautiful Themes**: Choose from 4 professionally designed themes (Playful Pastel, Elegant Dark, Warm Mediterranean, Minimal Zen)
- **Shareable Links**: Each journey gets a unique URL to share with recipients
- **Mobile-First Design**: Fully responsive experiences that work on any device
- **Multi-Language Support**: Available in English and German
- **Free Forever**: No payment required, unlimited journeys

## How It Works
1. **Sign up** with email (passwordless magic link authentication)
2. **Describe your gift** to the AI chat interface
3. **AI generates** a complete journey tailored to your occasion
4. **Customize** in the visual editor (reorder, edit text, add images, change theme)
5. **Publish** to generate a shareable link
6. **Share** the link with your recipient
7. **Recipient experiences** the journey on their device, answering questions and discovering reveals

## Technical Details
- Built with Next.js 16 and React 19
- Powered by Anthropic Claude AI (Sonnet 4.5)
- Hosted on Vercel
- Free tier includes unlimited journeys and storage

## Screen Types Available
1. **Hero Screen**: Opening story moment with headline, body text, and optional background image
2. **Choice Screen**: Multiple choice questions (2-4 options, single or multi-select)
3. **Text Input Screen**: Open-ended question where recipient types their answer
4. **Reveal Screen**: Big reveal moment with confetti animation and call-to-action button
5. **Media Screen**: Full-screen image display with optional caption
6. **End Screen**: Closing screen (automatically added)
